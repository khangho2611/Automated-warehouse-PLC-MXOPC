# I/O Mapping
This file documents the signal mapping between Factory I/O Warehouse, MX OPC, and GX Works3 PLC.
| Factory I/O Device | PLC Address | Type | Description |
|---|---|---|---|
| Sensor 1 | X0 | Input | Detect pallet/object |
| Conveyor Motor | Y0 | Output | Run conveyor |
| Lift Up | Y1 | Output | Move lift upward |
| Lift Down | Y2 | Output | Move lift downward |
| Stop/Reset | M0 | Internal Bit | Reset sequence |
